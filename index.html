HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element is self explanatory.
  • The heading elements (h1-h6) are used to represent headings of sections, with h1 being the highest level, h6 the lowest.
  • The paragraph (p) element represents a paragraph.
  • The anchor (a) element signifies a link, often used for navigation.
  • The list (ul, ol) elements represent an unordered and ordered list
  • The list item (li) element is an individual item within either of the aforementioned lists

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color applies to text
  • Background-color applies to background
  • Px is a common unit of measurement for element size

Git

  • git status: checks what branch you are on.
  • git checkout -b branch-name: forks off a branch and focuses on it
  • git add -A: stage all files for commit
  • git commit -m comment: commits files with the comment

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A variable must be defined before it is used in a function.
